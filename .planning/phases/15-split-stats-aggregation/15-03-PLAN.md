Extend `PlayerMatch` with:
```ts
outcome: 'won' | 'lost' | 'draw' | null
```

When hydrating completed matches:
- if there are no result rows, leave `outcome` and `won` as `null`
- if all normalized result rows are won by the player's team, set `outcome = 'won'`
- if all normalized result rows are won by the opposing team, set `outcome = 'lost'`
- if the result rows split between both teams, set `outcome = 'draw'`

Keep `won` for compatibility:
- `true` for `outcome === 'won'`
- `false` for `outcome === 'lost'`
- `null` for `outcome === 'draw'` or no result
