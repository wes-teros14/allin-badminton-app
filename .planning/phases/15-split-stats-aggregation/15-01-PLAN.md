Recommended contract:
```ts
export interface MatchPlayersLike {
  team1_player1_id: string
  team1_player2_id: string
  team2_player1_id: string
  team2_player2_id: string
}

export function computeStatsFromResults(
  match: MatchPlayersLike & { match_results: MatchResultLike[] | null | undefined }
): Map<string, { wins: number; games: number }>
```

For each normalized result row:
- increment `games` for all four players
- increment `wins` for the two players on the winning pair

- one legacy row -> winners 1 win / 1 game, losers 0 wins / 1 game
- `2-0` split -> winners 2 wins / 2 games, losers 0 wins / 2 games
- `1-1` split -> every player 1 win / 2 games
