**Hook implementation steps:**

1. **Fetch batches ordered by cost_per_tube ASC** (cheapest first, per D-04):
```typescript
const { data: batchRows, error } = await supabase
  .from('shuttle_batches')
  .select('id, brand, tube_count, cost_per_tube, notes, purchased_at, created_at')
  .order('cost_per_tube', { ascending: true })
  .order('created_at', { ascending: true })
```

2. **Fetch all shuttle_usage rows** (to compute remaining per batch):
```typescript
const { data: usageRows } = await supabase
  .from('shuttle_usage')
  .select('batch_id, tubes_used')
```

3. **Compute tubes_remaining per batch** using a Map:
```typescript
const usageMap = new Map<string, number>()
for (const u of (usageRows ?? [])) {
  usageMap.set(u.batch_id, (usageMap.get(u.batch_id) ?? 0) + u.tubes_used)
}
```

4. **Compute tube_start/tube_end ranges** — these are determined by the ORDER of batches sorted by `created_at ASC` (creation order, not cost order). The first batch ever created gets T-1001. So:
   - Fetch a SECOND query of batches ordered by `created_at ASC` to assign sequential tube IDs
   - Compute cumulative offset: batch[0].tubeStart = 1001; batch[1].tubeStart = batch[0].tubeStart + batch[0].tube_count; etc.
   - Store the tubeStart in a Map<id, number> keyed by batch id

```typescript
const { data: orderedByCreation } = await supabase
  .from('shuttle_batches')
  .select('id, tube_count')
  .order('created_at', { ascending: true })

const tubeStartMap = new Map<string, number>()
let cursor = 1001
for (const b of (orderedByCreation ?? [])) {
  tubeStartMap.set(b.id, cursor)
  cursor += b.tube_count
}
```

5. **Map to ShuttleBatch[]** using the cost-ordered batchRows:
```typescript
const batches: ShuttleBatch[] = (batchRows ?? []).map((b) => {
  const tubesUsed = usageMap.get(b.id) ?? 0
  const tubeStart = tubeStartMap.get(b.id) ?? 1001
  return {
    id: b.id,
    brand: b.brand,
    tubeCount: b.tube_count,
    costPerTube: Number(b.cost_per_tube),
    notes: b.notes,
    purchasedAt: b.purchased_at,
    createdAt: b.created_at,
    tubesRemaining: Math.max(0, b.tube_count - tubesUsed),
    tubeStart,
    tubeEnd: tubeStart + b.tube_count - 1,
  }
})
```

6. **addBatch mutation** — inserts into shuttle_batches, requires user.id as created_by:
```typescript
async function addBatch(input: AddBatchInput): Promise<{ error: string | null }> {
  if (!user) return { error: 'Not authenticated' }
  const { error } = await supabase.from('shuttle_batches').insert({
    brand: input.brand,
    tube_count: input.tubeCount,
    cost_per_tube: input.costPerTube,
    notes: input.notes ?? null,
    created_by: user.id,
  })
  if (error) return { error: error.message }
  await fetchBatches()
  return { error: null }
}
```

7. **totalStockRemaining** computed from batches:
```typescript
const totalStockRemaining = batches.reduce((sum, b) => sum + b.tubesRemaining, 0)
```
